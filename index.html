Sincronização na nuvem

Ligue uma base de dados gratuita (Firebase) para usar o aplicativo em vários telemóveis e computadores, em tempo real.

Configuração (uma só vez):
1. Aceda a console.firebase.google.com e crie um projeto gratuito.
2. Build → Firestore Database → Criar base de dados.
3. Build → Authentication → Sign-in method e ative Anónimo.
4. Ícone </> (Web app) → copie o objeto firebaseConfig e cole abaixo.
5. Toque em Ativar. Repita o passo 4–5 em cada dispositivo (ou aloje o ficheiro num endereço).
